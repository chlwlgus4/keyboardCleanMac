KeyboardClean

개인정보 처리방침

KeyboardClean("앱")은 사용자의 키보드 청소 중 실수 입력을 방지하기 위해 일시적으로 키보드 입력을 차단하는 macOS 유틸리티입니다. 본 개인정보 처리방침은 앱이 어떤 정보를 수집하고 어떻게 처리하는지 설명합니다.

시행일: 2026년 3월 17일

문의: chlwlgus91@gmail.com

1. 수집하는 정보

현재 버전의 KeyboardClean은 사용자의 개인정보를 수집하지 않습니다.

  • 회원가입, 로그인, 프로필 정보 수집 기능이 없습니다.
  • 이름, 이메일, 연락처, 위치, 결제 정보, 건강 정보 등의 개인정보를 수집하지 않습니다.
  • 분석, 추적, 광고 SDK를 사용하지 않습니다.
  • 사용자 데이터를 외부 서버로 전송하지 않습니다.

2. 기기 권한 사용

앱은 키보드 입력을 차단하는 기능을 제공하기 위해 macOS의 손쉬운 사용(Accessibility) 권한을 요청할 수 있습니다. 이 권한은 오직 사용자가 청소 모드를 켰을 때 키 입력을 감지하고 차단하기 위해 로컬 기기 내에서만 사용됩니다.

  • 권한 정보는 개발자 서버로 전송되지 않습니다.
  • 사용자의 키 입력 내용은 저장되거나 전송되지 않습니다.

3. 로컬 저장 정보

앱은 사용자 편의를 위해 사용자가 설정한 종료 단축키와 일부 UI 상태를 기기 내부에만 저장할 수 있습니다. 이 정보는 macOS의 로컬 설정 저장소(UserDefaults)에 저장되며, 외부로 전송되지 않습니다.

4. 제3자 제공

앱은 개인정보를 수집하지 않으므로, 제3자에게 제공하거나 판매하지 않습니다.

5. 아동의 개인정보

앱은 아동의 개인정보를 포함한 어떠한 개인정보도 수집하지 않습니다.

6. 개인정보 처리방침 변경

앱의 기능이 변경되어 개인정보 처리 방식이 달라질 경우, 본 페이지를 통해 개인정보 처리방침을 업데이트할 수 있습니다. 향후 광고, 분석 도구, 계정 기능 또는 서버 통신이 추가되는 경우 본 방침도 함께 수정됩니다.

본 방침은 현재 공개된 KeyboardClean 버전에 적용됩니다.
